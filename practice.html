PIXTECHの思い

プログラミングの仕事を楽しむために

PIX TECH(ピクステック)のカリキュラムは、大手企業を経験した現役のプログラマーが教える、現場で活躍できるプログラマーを目指し、実際に現場に入って仕事ができるようになるところまでをゴールとしたカリキュラムです。

一人一人に質の高いサポートをするため少人数制を取っています。

現場に入ってしまえばそこはもう学校ではなく、仕事場です。確かに仕事を教えてくれる先輩はいるかもしれませんが、その先輩にも自分の業務があり、常に面倒を見てくれるわけもなく、もっというと必ずしも優しく仕事を教えてくれる先輩に出会えるかどうかはわかりません。

プログラマーはプログラミングスクールで学ぶことよりも、現場で学ぶことの方がたくさんあります。ですが多くの方はそれを知らずに現場に入り、なかなか成長できない自分に悩み、あんなに憧れていたプログラマーの仕事が楽しくなくなってしまいます。

PIX TECHでは卒業してからも現場で使い続けていける、「スキルの身につけ方」、「仕事の仕方」を習得しどんどん成長していける、もっと仕事が楽しめるようなプログラマーをたくさん育てて行きたいと思っています。

学べる言語

HTML(Hyper Text Markup Language)とはホームページを作るための基本的なマークアップ言語です。ホームページの骨組みの部分を作成します。

CSS(Cascading Style Sheets、カスケーディング・スタイル・シート)とは、webページのスタイル・デザインを指定するための言語です。レイアウトや背景などを作成します。

Java Script(ジャバスクリプト)とは、WEB上での動きのある表現をするために開発された簡易プログラミング言語です。ウェブページの様々な機能や動きを作成します。
